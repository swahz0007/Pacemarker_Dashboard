# 产品目标与范围

## 1. 产品定位

Pacemaker Dashboard 是面向 CIED 程控报告的数据工程与研究审阅系统。它把多品牌、多模板的 Excel 程控报告转换为结构化、可审计、患者级纵向数据，并通过仪表盘支持检索、概览和记录复核。

仪表盘是数据审阅与导航界面，不是论文研究对象，也不是临床决策系统。

## 2. 主要用户

- 数据管理员：运行提取、匹配、分组和增量更新。
- 研究人员：查看队列结构、字段覆盖、随访历史和脱敏演示。
- 审计人员：核对源 Excel 与结构化输出的一致性。
- 前端维护者：迭代仪表盘、响应式布局和发布流程。

## 3. 核心能力

1. 支持 `.xls` / `.xlsx` 多品牌报告处理。
2. 按模板和内容特征匹配报告结构。
3. 提取 header、程控参数、测量值、阈值、事件、结论和日期。
4. 按登记号生成患者级纵向记录。
5. 隔离身份冲突记录，避免静默归并。
6. 通过独立审计器进行字段级核验。
7. 生成匿名化公开数据和隐私审计报告。
8. 提供研究概览、患者详情、程控历史和图表分析。

## 4. 明确不做

- 不替代原始程控报告。
- 不生成诊断、处方或治疗建议。
- 不对电池寿命作无型号依据的通用临床判断。
- 不手工补造临床结局。
- 不覆盖原始数据或私有患者记录。
- 不把公开演示数据当作正式论文分析数据。

## 6. 成功标准

- 数据提取可重复运行且不会静默覆盖身份冲突。
- 每个公开发布包在上传前通过隐私审计。
- 前端在宽屏、常规桌面和平板/手机上无内容遮挡。
- 线上资源版本与本地发布记录可追踪。
- 研究数字可从冻结的结构化快照复现。
